# Demos

Use these runnable examples when you want one focused concern at a time instead
of the longer guided walkthroughs.

- [py-binary](../example/py-binary/README.md) - verify binary negotiation
- [py-http-cookie-auth](../example/py-http-cookie-auth/README.md) - pull a session cookie into `@auth_` at the
  HTTP boundary
- [py-links](../example/py-links/README.md) - return a prepopulated function-type link
- [py-select](../example/py-select/README.md) - select just the `id` field from a list of users
- [py-websocket](../example/py-websocket/README.md) - use WebSocket request/reply

Related guides:

- Start with the [Quickstart](./example.md) if you want the shortest end-to-end path.
- See the [Transport Guide](./03-build-clients-and-servers/01-transports.md) for the HTTP and WebSocket patterns
  behind these examples.
- See the [Auth Guide](./03-build-clients-and-servers/05-auth.md) for the recommended production version of the
  cookie-auth pattern.
