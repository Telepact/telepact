# Learn Telepact by Example

Let's learn Telepact one small step at a time.

Each page in this directory is intentionally standalone. That is why the setup
commands repeat: we never have to assume a previous example is still running, or
that we already have the Telepact repository checked out locally.

We'll use the published CLI from PyPI throughout:

```sh
uv tool install --prerelease=allow telepact-cli
```
