# Examples

Runnable end-to-end examples live in this directory:

- [py-binary](./py-binary/README.md) - Minimal Python example that shows how to verify binary negotiation
- [py-http-cookie-auth](./py-http-cookie-auth/README.md) - Minimal Python example that shows how to pull a session cookie into `@auth_` at the HTTP boundary
- [py-links](./py-links/README.md) - Minimal Python example that shows how to return a prepopulated function-type link
- [py-mock-server](./py-mock-server/README.md) - Minimal Python example that shows how to call a mock Telepact server
- [py-select](./py-select/README.md) - Minimal Python example that shows how to select just the `id` field from a list of users
- [py-websocket](./py-websocket/README.md) - Minimal Python example that shows how to use WebSocket request/reply
- [py-websocket-request-ids](./py-websocket-request-ids/README.md) - Minimal Python example that shows how to match parallel WebSocket responses by `@id_`
