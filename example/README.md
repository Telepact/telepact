# Demos

Runnable end-to-end demos live in these GitHub folders:

- [py-binary](./py-binary/) - Minimal Python demo that shows how to verify binary negotiation
- [py-http-cookie-auth](./py-http-cookie-auth/) - Minimal Python demo that shows how to pull a session cookie into `@auth_` at the HTTP boundary
- [py-links](./py-links/) - Minimal Python demo that shows how to return a prepopulated function-type link
- [py-select](./py-select/) - Minimal Python demo that shows how to select just the `id` field from a list of users
- [py-websocket](./py-websocket/) - Minimal Python demo that shows how to use WebSocket request/reply
