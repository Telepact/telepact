# Demos

Runnable end-to-end demos live in these GitHub folders:

- [node-browser-docker](https://github.com/Telepact/telepact/tree/main/example/node-browser-docker) - Canonical browser + Node + Docker example with a browser fetch client, Node gateway, backing Telepact service, cookie auth bridging, binary negotiation, `@select_`, function links, and CLI schema fetch/compare
- [py-binary](https://github.com/Telepact/telepact/tree/main/example/py-binary) - Minimal Python demo that shows how to verify binary negotiation
- [py-http-cookie-auth](https://github.com/Telepact/telepact/tree/main/example/py-http-cookie-auth) - Minimal Python demo that shows how to pull a session cookie into `@auth_` at the HTTP boundary
- [py-links](https://github.com/Telepact/telepact/tree/main/example/py-links) - Minimal Python demo that shows how to return a prepopulated function-type link
- [py-select](https://github.com/Telepact/telepact/tree/main/example/py-select) - Minimal Python demo that shows how to select just the `id` field from a list of users
- [py-websocket](https://github.com/Telepact/telepact/tree/main/example/py-websocket) - Minimal Python demo that shows how to use WebSocket request/reply
